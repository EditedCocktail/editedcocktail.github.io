Installation Tutorial

  1. Close VoiceCatX (if open)
  2. Create an empty folder or go to your VoiceCatX folder
  3. Add this folder to Windows Defender exclusions
  4. Place VoiceCatXManager.exe in this folder
  5. Run it
  6. If the folder is empty, clicking the button will install VoiceCatX in this folder
  7. If VoiceCatX is already installed in this folder, it will check for a newer version and update if available
